# File Report: .htaccess

## Purpose
Apache server configuration for the root directory.

## Overview
Prevents direct access to dotfiles for security and redirects all requests not starting with `/public/` to the `public/` directory.

## File Location
`.htaccess`

## Rules
- Denies access to files starting with a dot (`.`).
- Redirects `^(.*)$` to `public/$1`.
