# Dosya Raporu: config/.htaccess

## Amaç
`config/` dizini için güvenlik yapılandırması.

## Genel Bakış
Yapılandırma dosyalarının ve hassas verilerin ifşa edilmesini önlemek için `config/` dizinine tüm doğrudan HTTP erişimlerini reddeder.

## Dosya Konumu
`config/.htaccess`

## Kurallar
- `Require all denied`

## Kaynak Referansları
- `config/.htaccess:1-4`
