# Dizin Raporu: storage

## Amaç
Uygulama durumu ve tanılama için yazılabilir dizinleri içerir.

## Alt Dizinler
- `cache/`: Uygulama ve görünüm önbelleği.
- `logs/`: Uygulama günlük dosyaları.
- `profiler/`: Hata ayıklama araç çubuğu için JSON profil anlık görüntüleri.

## Kaynak Dosyaları
- `.htaccess`: Erişim kısıtlama kuralları.

## Kaynak Referansları
- Dizin: `storage/`
