# Dosya Raporu: core/.htaccess

## Amaç
`core/` dizini için güvenlik yapılandırması.

## Genel Bakış
Çerçeve (framework) kaynak kodunun ifşa edilmesini önlemek için `core/` dizinine olan tüm doğrudan HTTP erişimini engeller.

## Dosya Konumu
`core/.htaccess`

## Kurallar
- `Require all denied`

## Kaynak Referansları
- `core/.htaccess:1-4`
