# Dizin Raporu: public

## Amaç
Uygulamanın dışarıdan erişilebilen tek dizinidir.

## Alt Dizinler
- `css/`: Statik stil dosyaları.
- `uploads/`: Kullanıcı tarafından yüklenen dosyalar.

## Kaynak Dosyaları
- `.htaccess`: Apache yeniden yazma (rewrite) kuralları.
- `index.php`: Ön Kontrolcü (Front Controller).

## Kaynak Referansları
- Dizin: `public/`
