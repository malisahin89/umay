# File Report: public/.htaccess

## Purpose
Apache configuration for URL rewriting and security.

## Overview
Configures the web server to redirect all non-existent files and directories to `index.php` (Front Controller pattern) and ensures the `Authorization` header is passed to PHP.

## File Location
`public/.htaccess`

## Implementation
- `RewriteEngine On`: Enables the rewrite engine.
- `Authorization Header`: Forces the `Authorization` header into an environment variable `HTTP_AUTHORIZATION` to prevent it from being stripped by Apache/CGI.
- `Frontend Controller`: Redirects all requests that are not existing files to `index.php`.
