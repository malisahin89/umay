# Dosya Raporu: public/.htaccess

## Amaç
URL yeniden yazma (rewriting) ve güvenlik için Apache yapılandırması.

## Genel Bakış
Web sunucusunu, mevcut olmayan tüm dosya ve dizinleri `index.php`'ye (Front Controller deseni) yönlendirecek şekilde yapılandırır ve `Authorization` başlığının PHP'ye iletilmesini sağlar.

## Dosya Konumu
`public/.htaccess`

## Uygulama
- `RewriteEngine On`: Yeniden yazma motorunu etkinleştirir.
- `Authorization Header`: Apache/CGI tarafından kaldırılmasını önlemek için `Authorization` başlığını zorunlu olarak `HTTP_AUTHORIZATION` ortam değişkenine atar.
- `Frontend Controller`: Mevcut olmayan tüm istekleri `index.php`'ye yönlendirir.

## Kaynak Referansları
- `public/.htaccess:1-15`
