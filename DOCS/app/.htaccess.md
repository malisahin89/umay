# Dosya Raporu: app/.htaccess

## Amaç
`app/` dizini için güvenlik yapılandırması.

## Genel Bakış
Uygulama kaynak kodunun ifşa edilmesini önlemek için `app/` dizinine olan tüm doğrudan HTTP erişimini engeller.

## Dosya Konumu
`app/.htaccess`

## Kurallar
- `Require all denied`

## Kaynak Referansları
- `app/.htaccess:1-4`
