# Dosya Raporu: database/.htaccess

## Amaç
Veritabanı dizini için güvenlik yapılandırması.

## Genel Bakış
Migrasyon dosyalarının ve seeder'ların ifşa edilmesini önlemek amacıyla, derinlemesine savunma (defense-in-depth) önlemi olarak `database/` dizinine tüm doğrudan HTTP erişimlerini engeller.

## Dosya Konumu
`database/.htaccess`

## Uygulama
- `Require all denied`: Bu dizine yapılan tüm doğrudan HTTP isteklerini açıkça reddeder.

## Kaynak Referansları
- `database/.htaccess:1-4`
